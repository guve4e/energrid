# Installation Passport

The Installation Passport is Energrid's structured record for a customer electrical installation.

It replaces scattered paper notes, OneNote pages, photos, and memory with one durable installation record.

## Purpose

Energrid does not only install cables and breakers. Energrid leaves behind documented infrastructure.

The Installation Passport gives each project:

- circuit schedule
- panel information
- cable and breaker data
- installation notes
- service history
- future upgrade path
- optional QR code access from the panel

## MVP Scope

The first version should stay simple:

- create installation
- add panel
- add circuits
- add service entries
- view record internally

Do not start with:

- floor plans
- public customer portal
- QR auth
- PDF generation
- photo storage
- AI diagnostics

Those can come later.

## Strategic Value

This is a differentiator against low-trust local electricians.

A normal electrician finishes the job and disappears.

Energrid leaves a documented system that can be inspected, maintained, upgraded, and understood years later.
